Gemini nutzen um eine optimale installationsanweisung für eine Gentoo Hardend Installation zu bekommen 

1. API Key erstellen auf Google AI Studio
2. export GEMINI_API_KEY="AIzaSy..." - "Dein API key in die Ausrufezeichen einfügen" oder die py bearbeiten Zeile if name == "main": key = os.getenv("GEMINI_API_KEY")
3. Nutzen zb #python3 ~/Downloads/gemini_bootstrap.py

Die datei am besten mit auf einen multiboot usb stick packen ist die einfachste lösung.
